Title: Notepad Website

Description:

The Notepad Website is a responsive frontend project built
using HTML, CSS, and JavaScript. It provides a simple and
user-friendly interface for creating, editing, and managing
notes directly in the browser.


Vision:

The main goal of this project is to improve frontend
development skills by creating a professional Notepad
website. It demonstrates how HTML, CSS, and JavaScript
work together to build an interactive web application.


Technologies Used:

1. html5
2. CSS3
3. JavaScript (ES6)


Features:

1. Professional Header Section
2. Two Navigation Bars
3. File, Edit and View Dropdown Menus
4. Note Title Input
5. Large Writing Area
6. Responsive Layout
7. Hover Effects
8. Status Bar
9. Word Counter
10. Character Counter
11. Line Counter
12. Column Counter
13. Save Note Functionality
14. Clear Note Functionality
